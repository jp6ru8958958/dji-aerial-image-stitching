# dji-aerial-image-stitching
Create your virtual environment and enable it:
```
virtualenv -p ~/.pyenv/versions/3.6.12/bin/python venv
. venv/bin/activate
pip install -r requirements.txt
```
Run:
```
python main.py data/test
```
